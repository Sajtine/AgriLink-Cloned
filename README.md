# AgriLink: Empowering Farmers through Smart Technology
**AgriLink** is an Android application built for farmers to **connect directly with local vendors** to sell agricultural products and to receive **real-time weather guidance**.  
It’s designed to connect farmers to **marketplace connection** to boost their productivity and income.

### Farmer-Vendor Marketplace
- Farmers can **browse vendors near their location**
- Farmers can **send product sell requests to vendors**
- Promotes **direct selling**, reducing dependency on middlemen


## Built With

- **Java** – Core language for Android  
- **Android Studio** – Development IDE  
- **Retrofit** – API networking  
- **OpenWeatherMap API** – Weather data provider
